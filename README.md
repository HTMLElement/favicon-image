# favicon-image

Favicon web component that works across devices and resolutions.

## Install
```bash
bower install --save HTMLElement/favicon-image
```

## Usage
```html
<html>
  <head>
    <link rel="import" href="favicon-image.html">
  </head>
</html>
```
